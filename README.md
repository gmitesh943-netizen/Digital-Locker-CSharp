# 🚀 Digital Locker System (C# .NET)

A Windows Forms based **Digital Locker Application** built using C# and .NET that allows users to securely upload and manage files.

## 🔹 Features

* User Registration & Login System
* Secure Authentication
* File Upload Functionality
* Image Preview Before Upload
* File Path Storage
* Upload Date & Time Tracking
* Simple & User-Friendly UI
